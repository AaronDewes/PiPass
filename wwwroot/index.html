Thank you for using PiPass for the Raspberry Pi 2 Model B! PiPass is a project designed around the notion of "Homepass" for the Nintendo 3DS family. It allows you to streetpass others by using Nintendo Zones in the comfort from your home.

This project was inspired by Spillpass. Check it out as the developer has a working solution for the Raspbery Pi Model B and is working on a solution for the Raspberry Pi 2 Model B.

PiPass has not been tested on other versions of Raspberry Pi, so use at your own risk. I've commented the project quite well, so anyone can feel free to port it over.

By default, PiPass automatically starts up with your Raspberry Pi. The PiPass Dashboard is your central hub to control PiPass; however, you will need to manually edit current_zones.xml to make changes to the list of Nintendo Zones. I will look into adding a feature to edit current_zones.xml through the PiPass Dashboard. This XML file can be found here: /wwwroot/assets/xml/current_zones.xml.

Any updates to PiPass can be found on my GitHub. Also, please feel free to comment on the project with any feature requests or bug fixes. I will try to make some time to address any issues.

Again, thank you for using PiPass!
