About

Thank you for using PiPass for the Raspberry Pi! PiPass turns your Raspberry Pi into a Nintendo 3DS Homepass Relay Station. The PiPass Dashboard can be accessed through a computer or mobile device. It can manage features without the need of opening up any files manually. However, some fine tuning may require getting your hands 'dirty'. Though the process is relatively fast and easy.

Updates to PiPass and references in configuring PiPass can be found on my GitHub. Please feel free to report any bugs or request any features that you might want.

Thank you!
